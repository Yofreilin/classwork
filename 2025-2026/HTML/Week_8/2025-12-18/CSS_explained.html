CSS explained

Term Definitions
CSS attribute: display values: box, inline, flex, grid, none The CSS display property is a fundamental attribute that controls how an HTML element is rendered and how it interacts with other elements in the document's flow. It determines the type of box an element generates, which is crucial for page layout.
CSS attribute: font-size he font-size CSS property sets the size of the font. Changing the font size also updates the sizes of the font size-relative units, such as em, ex, and so forth.
CSS size font-relative units: em, rem, ch, cap, ex, lh, ic a type of relative length unit that define a size based on the dimensions of the browser's viewport
CSS size viewport relative units: vh, vw, vmax, vmin (viewport height): 1vh equals 1% of the height of the viewport.
CSS absolute units Absolute units do not change with the screen and are not recommended for websites
CSS attribute: font-family The CSS font-family property specifies the prioritized list of typefaces (fonts) to be used for the selected element's text.
CSS margin The CSS margin property defines the transparent space around an element, outside of any defined borders.
CSS border The CSS border properties are used to define the style, width, and color of an element's border, which is a line that wraps around the element's padding and content.
CSS padding CSS padding is the space between an element's content (text, images, etc.) and its border.
